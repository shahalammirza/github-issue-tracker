1. What is the difference between var, let, and const?
var: Function scoped variable, can be re-declared and updated.
let: Block scoped variable, can be updated but cannot be re-declared in the same scope.
const: Block scoped variable, cannot be updated or re-declared.

2. What is the spread operator (...)?
Ans: The spread operator ... is used to expended elements of an array or object into another array or object.

3. What is the difference between map(), filter(), and forEach()?
map(): Transforms each elements and returns a new array.
filter(): Select elements based on a condition and returns a new array.
forEach() : Executes a function for each element but does not return a new array.

4. What is an arrow function?
Ans: An arrow function is a shorter syntax for writing function in javascript.

5. What are template literals?
Ans: Template literals allow you to embed variables or expression inside a string using backticks ``.
